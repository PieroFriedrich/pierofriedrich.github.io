Game Project

Word Guesser

Word Guesser consists of a game where the player needs to guess the right word of the day, chosen randomly by an API. The player has 6 attempts to choose a valid English word and receive different colors as hints for the next turn. The game was inspired in the famous Wordle game, currently owned by The New York Times. It can be played clicking on this link.

Technologies used include:

  • HTML
  • CSS
  • JavaScript
